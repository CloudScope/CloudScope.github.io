Production EKS Cluster on AWS

A fully automated, production-grade Amazon EKS cluster provisioned with Terraform. Includes multi-AZ node groups, managed add-ons (CoreDNS, kube-proxy, VPC CNI), IRSA-based pod IAM roles, cluster autoscaler, and ALB Ingress Controller.

  • Infra: VPC with public/private subnets across 3 AZs, NAT Gateway, bastion host
  • EKS: Managed node groups with launch templates, taints, and spot instance support
  • IAM: IRSA (IAM Roles for Service Accounts) for fine-grained pod permissions
  • Networking: AWS ALB Ingress with IngressGroup for shared load balancer
  • Autoscaling: Cluster Autoscaler + HPA configured for production workloads

Jenkins CI/CD Pipeline for Microservices

End-to-end CI/CD pipeline using Jenkins for a microservices architecture deployed on Kubernetes. Implements GitOps principles with Argo CD, automated Docker image builds, semantic versioning, and environment promotion from dev → staging → prod.

Terraform AWS Landing Zone

Multi-account AWS landing zone using Terraform modules. Sets up AWS Organizations, centralised logging, GuardDuty, Security Hub, and networking (Transit Gateway + VPC peering) across dev, staging, and prod accounts from a single codebase.

  • Accounts: AWS Organizations with SCPs, delegated admin for Security Hub
  • Networking: Transit Gateway hub-and-spoke across accounts and regions

  • State: Remote state in S3 + DynamoDB locking with per-account workspaces

⚡ GitHub Actions ☁ AWS 🏗 Terraform
Active

GitHub Actions Self-Hosted Runner on ECS

Auto-scaling GitHub Actions self-hosted runners using ECS Fargate. Runners spin up on demand, execute the workflow job, and terminate — eliminating idle costs and fixing the cold-start problem of queued jobs.

  • Runners: ECS Fargate tasks registered as GitHub Actions runners via API token
  • Scaling: EventBridge + Lambda watches job queue; scales ECS service up/down
  • Security: Runners in private subnets, IRSA for AWS access, no persistent credentials
  • Cost: ~$0 when idle — Fargate tasks terminate after job completion
  • IaC: Full Terraform module — ECS cluster, task definition, IAM, EventBridge rules
